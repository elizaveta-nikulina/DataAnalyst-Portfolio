## Описание
В рамках проекта был создан дашборд на основе данных о конференциях TED. 
Проект включал анализ и визуализацию информации о выступлениях, темах и лекторах конференций. 

## Используемые инструменты
- Yandex DataLens
- Визуализация данных
- Дашборды
